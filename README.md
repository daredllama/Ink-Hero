## Description
My project is a 2D puzzle game called Ink Hero. You play as a sentient ink blob whose job it is to save the world by drawing pictures of Christmas. Why? Because an evil wizard made everyone forget everything about Christmas, it is your job to remind everyone. This is my first-ever game!

## Playable
My game is on itch.io: https://daredllama.itch.io/inkhero
